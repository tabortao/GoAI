GoAI API Quick Start (Docker)

1. Start the Service (Docker Compose)

docker-compose up --build -d
Make sure config.json is properly configured.

2. Call the API

POST /api/v1/generate
{
  "prompt": "Translate the following text into Chinese:",
  "text": "Knowlegde can change your fate and English can accomplish your future",
  "model": "gpt-4o"
}
Example: Use curl to call the API
curl -X POST http://localhost:8080/api/v1/generate \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Translate the following text into Chinese:",
    "text": "Knowlegde can change your fate and English can accomplish your future",
    "model": "gpt-4o"
  }'
Or use PowerShell:
Invoke-RestMethod -Uri "http://localhost:8080/api/v1/generate" \
  -Method Post \
  -ContentType "application/json; charset=utf-8" \
  -Body (@{
    "prompt" = "Translate the following text into Chinese:"
    "text" = "Knowlegde can change your fate and English can accomplish your future"
    "model" = "gpt-4o"
  } | ConvertTo-Json -Compress)
